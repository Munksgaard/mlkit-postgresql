# mlkit-postgresql

Postgresql library for Standard ML / MLKit.

## Assumptions

A working MLKit installation (see
https://github.com/melsman/mlkit). Use `brew install mlkit` on macOS.

## Testing

To test the library, first do as follows:

    $ cd src
    $ make

Then, proceed as follows:

    $ cd ../test
    $ make init
    $ make
